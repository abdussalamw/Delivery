# 🚀 Smart Delivery System (نظام ديلفرو الذكي)

نظام إدارة توصيل تفاعلي واحترافي، صُمم لإدارة الطلبات، العملاء، والمناديب في الوقت الفعلي. يتميز النظام بلوحة تحكم بصرية حديثة، بالإضافة إلى **بوت واتساب آلي** مبني بـ Node.js/Baileys لاستقبال الطلبات من العملاء عبر رسائل الواتساب وحساب التكلفة تلقائياً وفقاً لعدد الأماكن المُرسلة.

---

## ✨ المميزات الرئيسية للطرح

* **📦 لوحة تحكم تفاعلية (KPIs):** مراقبة شاملة للمبيعات، المناديب النشطين، والطلبات الحالية.
* **🤖 أتمتة الواتساب (WhatsApp Bot):** استقبال الرسائل من العملاء، استخراج المواقع إذا تم إرسالها، وحساب رسوم التوصيل تلقائياً، وإضافة الطلب في قاعدة البيانات.
* **🛵 إدارة المناديب:** تتبع حالات المناديب (متاح، مشغول، في استراحة)، متابعة أدائهم وعدد الطلبات المُنفذة يومياً.
* **👥 إدارة العملاء:** حفظ تلقائي لبيانات أي عميل يراسل البوت لأول مرة، وإمكانية إضافة العملاء يدوياً، مع تتبع إجمالي طلباتهم والقيمة المادية.
* **🗺️ الخرائط:** التكامل السريع والانتقال نحو خرائط Google Maps استناداً للروابط المرسلة من العملاء.
* **🗃️ قاعدة بيانات متطورة:** يعتمد النظام على `PostgreSQL` التي تضمن أداء وحماية سريعين، مع نظام Views للحصول على الإحصائيات مباشرة.

---

## 🛠️ التقنيات المستخدمة

* **الواجهة الأمامية (Frontend):** HTML5, CSS3 (Glassmorphism UI), Vanilla JavaScript.
* **الخادم والأكواد (Backend):** PHP 8 (PDO).
* **قاعدة البيانات:** PostgreSQL 16.
* **بوت الواتساب:** Node.js, Express, @whiskeysockets/baileys.
* **مدير العمليات:** PM2.

---

## ⚙️ دليل التثبيت والتشغيل المحلي

### 1️⃣ إعداد قاعدة البيانات (PostgreSQL)
1. تأكد من تثبيت [PostgreSQL](https://www.postgresql.org/download/).
2. قم بإنشاء قاعدة بيانات جديدة باسم `delivery_db` في سيرفر `localhost`.
3. قم بتنفيذ الاستعلامات الموجودة في ملف `delivery_system_schema.sql` لبناء الجداول والـ Views وإضافة البيانات التجريبية.

### 2️⃣ إعداد خادم PHP (XAMPP)
1. تأكد من تفعيل إضافة PostgreSQL لـ PHP. افتح ملف `php.ini` وأزل الفاصلة المنقوطة (`;`) من السطور التالية:
   ```ini
   extension=pdo_pgsql
   extension=pgsql
   ```
2. ضع مجلد المشروع داخل مجلد `C:\xampp\htdocs\Delivery`.
3. راجع بيانات الاتصال بقاعدة البيانات في ملف `config.php` وتأكد من مطابقتها (اسم المستخدم، وكلمة المرور).

### 3️⃣ تشغيل بوت الواتساب (Node.js)
النظام يعتمد على بوت يستقبل الطلبات لتفعيل هذه الميزة اتبع التالي:
1. حمّل وثبّت [Node.js](https://nodejs.org/).
2. افتح شاشة الأوامر كمسؤول (Administrator Command Prompt).
3. ثبّت مدير العمليات PM2 عالمياً (سيبقي البوت يعمل بالخلفية دائمًا):
   ```bash
   npm install -g pm2
   ```
4. توجّه إلى مسار البوت وثبّت حزم المشروع:
   ```bash
   cd C:\xampp\htdocs\Delivery\bot
   npm install
   ```
5. قم بتشغيل الخادم الخاص بالواتساب من خلال PM2:
   ```bash
   pm2 start bot.js --name delivery-bot
   ```

### 4️⃣ الدخول والمتابعة
* يمكنك الآن الانتقال للصفحة الرئيسية: [http://localhost/Delivery/dashboard.php](http://localhost/Delivery/dashboard.php)
* لربط الواتساب بمسح رمز QR أو إدخال كود الجهاز، توجه لصفحة **⚙️ إعدادات الواتساب** في النظام حيث ستجد لوحة التحكم الكاملة.

## 🛡️ ملاحظات أمنية وهيكلية
* يحتوي المشروع على مسار خاص يُدعى `webhook_handler.php`. هذا المسار لا يجب الدخول إليه مباشرة من المتصفح بل يتواصل معه الـ Node.js bot محلياً.
* يُنصح بتغيير كلمات المرور الافتراضية لقاعدة البيانات عند الاستضافة على خوادم الإنتاج (VPS).
